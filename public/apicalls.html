API calls and examples

Add Transaction

Request URL: http://fetchrewards-api.herokuapp.com/api/add

This call is type "POST", receives JSON formatted data and this will add a transaction to our database

Example:
{ "payer": "DANNON", "points": 300, "timestamp": "2020-09-31T10:00:00Z" }

Returns:
{
"success": true,
"body": {
    "payer": "DANNON",
    "points": 300,
    "timestamp": "2020-09-31T10:00:00Z"
    }
}

Spend Points

Request URL: http://fetchrewards-api.herokuapp.com/api/spend

This call is type "POST", receives JSON formatted data and this will substract points from payers from oldest to newest from database

Example:
{"points": 5000}

Returns points substracted for each payer:
{
  "success": true,
  "body": [
   {
    "payer": "UNILEVER",
    "points": -2000
   },
   {
    "payer": "MILLER COORS",
    "points": -1000
   },
   {
    "payer": "DANNON",
    "points": -2000
   }
  ]
}

Point Balance

Request URL: http://fetchrewards-api.herokuapp.com/api/balance

This call is type "GET", and returns a balance of each payer's points.

Return:
{
  "success": true,
  "body": {
   "DANNON": 1000,
   "UNILEVER": 2300,
   "MILLER COORS": 100
  }
}

Errors

Errors will return the key "success" as "false" and include a key "message" with details.

Example:
{
  "success": false,
  "message": "Transaction invalid. Payer does not have enough balance"{
}
