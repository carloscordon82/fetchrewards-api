Download source code and run it locally

Step # 1 - Install NodeJS & NPM

Using a package (Mac & Windows):
Click the button below to go to NodeJS website and download the installer for your OS. This will install Node and NPM

NodeJS + NPM Download

Using Homebrew (Mac):
Type into your terminal: brew install node

Step # 2 - Source code

Click the button below and download the source code in a zip file, or clone it to your coding platform.

Github Repository

Step # 3 - Run it

Open your terminal and navigate to the root folder of the source code downloaded.
Type: `npm install` and press enter. Then type `npm run dev` and press enter.

Step # 4

Your terminal should show a message with the address where your server is running on, including the port number.
Use this URL to make all your API calls to.
